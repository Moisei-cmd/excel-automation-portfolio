# Автоматизация Excel на Python – моё портфолио

Я помогаю избавиться от ручной работы с Excel: собираю данные из многих файлов, анализирую таблицы любой структуры и заполняю шаблоны документов.

## Что внутри

### 1. Генератор отчёта с аналитикой (1_report_generator/report_generator.py)
Создаёт тестовые данные о продажах, считает выручку по товарам и дням, оформляет стильный отчёт. Демонстрирует полный цикл: генерация → анализ → оформление.

### 2. Универсальный анализатор (2_sales_analyzer/sales_analyzer.py)
Подстраивается под любые названия столбцов. Пользователь вводит, где цена, количество и что считать, а скрипт сам делает расчёт и сохраняет результат в Excel.

### 3. Заполнитель счёта с окошком (3_invoice_generator/invoice_gui.py)
Программа с графическим интерфейсом: выбираешь файл с товарами и шаблон счёта, нажимаешь кнопку – и получаешь заполненный документ с итогами.

## Как запустить
1. Установите Python 3 и зависимости:  
   `pip install pandas openpyxl pysimplegui`
2. Скачайте нужный скрипт.
3. Запустите через консоль: `python имя_скрипта.py`

## Стек
Python, pandas, openpyxl, PySimpleGUI, работа с файлами, защита от ошибок (try-except).
